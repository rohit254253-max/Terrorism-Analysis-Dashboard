# Terrorism-Analysis-Dashboard
Interactive terrorism analysis dashboard built using Python (Dash &amp; Plotly) featuring geospatial visualization and dynamic filtering for real-time insights.

## Features
- Interactive map visualization
- Country-based filtering
- Built using Dash and Plotly

## Upcoming Features
- Year filter
- Attack type filter
- Advanced analytics

This project visualizes global terrorism data using interactive maps and filters, enabling users to explore patterns across countries and time.
